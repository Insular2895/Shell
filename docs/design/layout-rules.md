# Layout rules

- Container max-width : `max-w-7xl mx-auto`
- Padding standard : `p-4 md:p-6`
- Gap grids : `gap-4` desktop, `gap-3` mobile
- Spacing tokens : Tailwind defaults (4px base)
- Border radius standard : `rounded-lg` (8px)
- Shadow par défaut : aucune (utiliser border)
