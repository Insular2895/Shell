# project-settings

Settings d'un projet/workspace :
- Tab General (nom, description)
- Tab Members (invite, roles)
- Tab Billing (lien vers Customer Portal Stripe)
- Tab API Keys (cf api-keys module)
- Tab Notifications
- Tab Danger Zone (delete)
Save partiel par tab.
