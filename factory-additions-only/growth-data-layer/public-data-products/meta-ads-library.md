# Meta Ads Library — produit data

## Source

Meta Ad Library API (publique, gratuite, RGPD-OK car publiée par les marques).

## Ce qu'on peut produire

- **Meta Ads Intelligence Report** (rapport concurrentiel, 50-300€)
  - Liste des creas actives d'un concurrent ces 30 derniers jours
  - Cadence de diffusion (créa nouvelle vs récurrente)
  - Angles marketing détectés
  - Landing pages utilisées
  - Catégories d'offres (promo, lead-gen, awareness)
  - Estimation budgetaire (si page > 25k followers, badge actif)

## Ce qu'on NE peut PAS produire

- ❌ Données utilisateur Meta (impossible techniquement et illégal)
- ❌ Audiences custom Meta (RGPD)
- ❌ "Quels users ont vu cette pub" (impossible et illégal)

## Respect TOS Meta

- Pas de scraping massif, on utilise l'API officielle
- Citation source obligatoire
- Pas de revente d'images de creas (droit d'auteur des marques)
- On vend l'analyse, pas les images
