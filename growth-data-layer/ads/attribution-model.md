# Attribution model — Ads

## MVP — first-touch
Visitor arrive avec UTM. Stockage cookie 30 jours. Toute conversion attribuée à premier UTM.

## Phase 2 — multi-touch
- Linear : crédit également réparti
- Time decay : récents pondérés davantage
- Position-based : 40% first + 40% last + 20% middle
- Data-driven : ML

Stack possible : Triple Whale, Northbeam, ou dbt custom.
