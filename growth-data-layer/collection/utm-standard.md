# UTM standards

| Param | Convention | Exemples |
|-------|-----------|----------|
| `utm_source` | provenance technique | `google`, `linkedin`, `partner-x`, `newsletter` |
| `utm_medium` | type de canal | `cpc`, `organic`, `email`, `affiliate`, `referral` |
| `utm_campaign` | nom de campagne | `2026-q1-launch`, `audit-promo` |
| `utm_term` | mot-clé | (CPC seulement) |
| `utm_content` | variation créative | `hero-blue`, `cta-primary` |

## Règles
- snake_case + tirets, pas d'espaces ni d'accents
- Documenter chaque campagne dans `tracking/campaign-tracking.md`
