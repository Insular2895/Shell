# admin-panel

Panel admin avec :
- Listing des users + filtres
- Détail user (edit, suspend, delete)
- Listing des subscriptions
- Logs récents
- Feature flags (toggle on/off par site)

Réservé aux users avec role='admin' (claim Supabase JWT).
