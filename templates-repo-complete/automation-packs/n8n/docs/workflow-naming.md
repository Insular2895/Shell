# Workflow naming convention

`<domain>-<action>[-<variant>]`

Exemples :
- `incident-auto-degrade`
- `pnl-daily-report`
- `quota-watcher`
- `lead-export-approval`
- `consent-audit-report`

Règles :
- snake-case avec tirets
- Domaine d'abord (ce qui le concerne)
- Verbe ensuite (ce qu'il fait)
- Pas plus de 30 caractères
- En anglais (cohérence avec n8n core)
